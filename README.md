# Password-strength-Checker
A Python-based password strength analyzer that evaluates complexity, length, and character diversity using clean, fundamental logic.
## 🚀 Features
* **Length Validation:** Checks if the password meets secure length requirements.
* **Character Diversity Scan:** Manually loops through inputs to verify uppercase, lowercase, numerical, and special character presence.
* **Strength Scoring:** Dynamic scoring system providing immediate feedback on whether a password is Weak, Medium, or Strong.
* **No External Dependencies:** Built entirely with native Python core logic.
